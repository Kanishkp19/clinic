## liveDoc

This project has been migrated from the original Gulp/Pug static workflow to Next.js.

### Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

### Production build

```bash
npm run build
npm run start
```

The original compiled theme assets remain in `public/` and are served directly by Next.js, preserving the existing UI and browser behavior.
